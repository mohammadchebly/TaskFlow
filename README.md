# Task Manager

A modern and responsive Task Manager web application built to help users organize, track, and manage their daily tasks efficiently.

## Features

- Add new tasks
- Edit and delete tasks
- Mark tasks as completed
- Responsive design
- Clean and modern UI
- Real-time updates using JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Purpose

This project was created to improve my frontend development skills and practice building interactive web applications using pure JavaScript.
